项目简介

水杉记言(Metasequoia Voice Input)是一个面向中文输入场景的开源语音输入工具，源代码在 GitHub 上可查看。本项目通过语音识别(ASR)将语音实时转换为文本，并可选接入文本润色能力，帮助你以更自然、更高效的方式完成输入。


当前设置窗口用于管理 ASR API、润色 API 和基础行为配置。您可以在“API 配置”和“基础设置”中修改参数，并通过“保存设置”提交到宿主程序。
